<p>SECTION: Systems &amp; Devices</p>
<p>SYSTEM: Printers</p>
<p>CODE: E02</p>
<p>Title</p>
<p>(E02 &mdash; Paper jam)</p>
<p>One-sentence summary</p>
<p>The printer detected a paper jam or paper feed obstruction and stopped the job.</p>
<p>What this code means</p>
<p>E02 is commonly used by some printer families to indicate a paper jam or paper feed problem. The exact jam location and sensor mapping depends on the model, but the general meaning is that paper did not move through the path as expected.</p>
<p>Where users usually see this code</p>
<p>Printer display panel when printing starts</p>
<p>After loading new paper or switching media types</p>
<p>Why this code usually appears</p>
<p>Paper is skewed, curled, or loaded incorrectly</p>
<p>Rollers are dirty or worn</p>
<p>Paper scraps remain from a previous jam</p>
<p>What typically happens next</p>
<p>The printer stops and prompts you to clear the jam</p>
<p>What this code is NOT</p>
<p>It is not a firmware crash</p>
<p>It is not a driver update problem</p>
<p>Troubleshooting checklist</p>
<p>Open access panels and remove jammed paper carefully</p>
<p>Remove all scraps; check the duplexer/rear path if equipped</p>
<p>Reload paper and align tray guides</p>
<p>Try a different paper type or a smaller stack</p>
<p>Related error codes</p>
<p>13.20 — Paper jam</p>
<p>6000 — Paper feed error</p>
<p>Notes and edge cases</p>
<p>Because code naming varies by manufacturer, confirm the code family for your exact printer model.</p>
