<p>SECTION: Systems &amp; Devices</p>
<p>SYSTEM: Printers</p>
<p>CODE: 13.20</p>
<p>Title</p>
<p>(13.20 &mdash; Paper jam)</p>
<p>One-sentence summary</p>
<p>A printer detected a paper jam condition in the paper path.</p>
<p>What this code means</p>
<p>13.xx error codes are widely used by many laser printer families to indicate paper jams. The specific suffix (such as 13.20) typically points to a jam location or sensor area, which varies by model.</p>
<p>Where users usually see this code</p>
<p>Printer control panel during printing</p>
<p>After a misfeed or skewed page</p>
<p>Why this code usually appears</p>
<p>Paper misfeed, skew, or torn scraps in the path</p>
<p>Worn rollers or pickup mechanisms</p>
<p>Incorrect paper type or tray guides</p>
<p>What typically happens next</p>
<p>The job stops and the printer prompts to clear the jam</p>
<p>What this code is NOT</p>
<p>It is not a firmware crash</p>
<p>It is not a network error</p>
<p>Troubleshooting checklist</p>
<p>Follow the printer’s on-screen jam removal prompts for the indicated area</p>
<p>Remove all paper scraps and verify sensors/doors close properly</p>
<p>Reload paper and ensure tray guides are aligned</p>
<p>Replace rollers/maintenance kit if jams are frequent</p>
<p>Related error codes</p>
<p>50.6 — Fuser error</p>
<p>49 Service Error — Firmware/service error</p>
<p>Notes and edge cases</p>
<p>Jam location codes differ by model; use the service manual for the exact jam area mapping.</p>
