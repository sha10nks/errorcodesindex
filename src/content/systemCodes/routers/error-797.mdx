<p>SECTION: Systems &amp; Devices</p>
<p>SYSTEM: Routers</p>
<p>CODE: Error 797</p>
<p>Title</p>
<p>(Error 797 &mdash; Modem not found)</p>
<p>One-sentence summary</p>
<p>The connection could not be made because the modem or network device was not found or not ready.</p>
<p>What this code means</p>
<p>Error 797 is a connection error indicating that the system could not find or use the required modem/network device for a dial/PPPoE-style connection. It can be caused by missing drivers, disabled adapters, or incorrect connection configuration.</p>
<p>Where users usually see this code</p>
<p>Broadband/PPPoE connections configured on a PC</p>
<p>After network adapter changes or driver updates</p>
<p>Why this code usually appears</p>
<p>Network adapter is disabled or driver is missing</p>
<p>Connection profile points to the wrong device</p>
<p>Device is unavailable due to system changes</p>
<p>What typically happens next</p>
<p>The connection fails immediately</p>
<p>What this code is NOT</p>
<p>It is not a password/authentication decline</p>
<p>It is not a DNS error</p>
<p>Troubleshooting checklist</p>
<p>Verify the network adapter is present and enabled</p>
<p>Update/reinstall the adapter drivers if needed</p>
<p>Recreate the connection profile and select the correct interface</p>
<p>Related error codes</p>
<p>Error 651 — Modem/router error</p>
<p>Error 769 — Destination not reachable</p>
<p>Notes and edge cases</p>
<p>On modern routers, PPPoE is usually configured on the router, not on each workstation.</p>
