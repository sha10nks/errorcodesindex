<p>SECTION: Systems &amp; Devices</p>
<p>SYSTEM: Smart Devices</p>
<p>CODE: 002</p>
<p>Title</p>
<p>(002 &mdash; Activation or setup error)</p>
<p>One-sentence summary</p>
<p>A Roku device could not complete setup/activation, commonly due to connectivity or service availability.</p>
<p>What this code means</p>
<p>Roku error code 002 is part of the setup/activation error family shown during device setup. It indicates the Roku could not complete the required activation step at that time.</p>
<p>Where users usually see this code</p>
<p>During device setup after a factory reset</p>
<p>When linking the device to an account</p>
<p>Why this code usually appears</p>
<p>Roku service is temporarily unavailable</p>
<p>Network connectivity or DNS issues</p>
<p>Router firewall/captive portal interference</p>
<p>What typically happens next</p>
<p>Setup stops until activation succeeds</p>
<p>What this code is NOT</p>
<p>It is not a channel-specific playback issue</p>
<p>It is not a Wi‑Fi password error by itself</p>
<p>Troubleshooting checklist</p>
<p>Restart Roku and router</p>
<p>Try a different network (mobile hotspot) to isolate home-network issues</p>
<p>Check whether captive portal sign-in is required on the network</p>
<p>Related error codes</p>
<p>001 — Activation or setup error</p>
<p>003 — Software update error</p>
<p>004 — Setup/activation error</p>
<p>Notes and edge cases</p>
<p>Some networks block device activation endpoints; testing on another network is a fast way to confirm.</p>
