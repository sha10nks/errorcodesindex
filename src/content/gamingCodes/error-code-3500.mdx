<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(Error Code 3500 &mdash; Unsupported region)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the service is not supported in the current region.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">Error Code 3500 appears when access is blocked due to regional availability rules. The restriction is policy-based. It does not indicate a technical error. Services remain inaccessible from unsupported locations. Availability may change over time.</span></p>

<p class="c1"><span class="c0">Content access attempts</span></p>
<p class="c1"><span class="c0">Login or feature entry</span></p>

<p class="c1"><span class="c0">Policy compliance rules</span></p>

<p class="c1"><span class="c0">Access remains blocked</span></p>
<p class="c1"><span class="c0">Content unavailable</span></p>
<p class="c1"><span class="c0">Status unchanged</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
