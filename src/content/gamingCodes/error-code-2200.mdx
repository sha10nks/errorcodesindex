<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that an active session has expired.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">Error Code 2200 appears when a session exceeds its allowed duration or becomes invalid. The expiration is enforced for security and consistency. It does not indicate account enforcement. Access ends until a new session is established. Session renewal is required.</span></p>
