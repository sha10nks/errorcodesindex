<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(Error Code 3600 &mdash; Authentication error)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that authentication could not be completed successfully.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">Error Code 3600 appears when authentication checks fail during access validation. The failure occurs before session establishment. It does not confirm account restriction. Access remains blocked until authentication succeeds. The issue may be temporary.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">Sign-in attempts</span></p>
<p class="c1"><span class="c0">Session creation</span></p>
<p class="c1"><span class="c0">Online feature access</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Authentication service interruption</span></p>
<p class="c1"><span class="c0">Validation failure</span></p>
<p class="c1"><span class="c0">Temporary system issue</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Login fails</span></p>
<p class="c1"><span class="c0">Access remains blocked</span></p>
<p class="c1"><span class="c0">Retry may succeed later</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a permanent ban</span></p>
<p class="c1"><span class="c0">It is not a server outage</span></p>
<p class="c1"><span class="c0">It is not a payment error</span></p>
