<p class="c1"><span class="c0">CODE: Error Code 700</span></p>
<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(Error Code 700 &mdash; Update required)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that an update is required before continuing.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">Error Code 700 appears when the service requires the client to be updated. The version in use is no longer accepted. This is a compatibility enforcement, not a system failure. Access is blocked until version requirements are met. The code does not indicate a problem with the account.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">Service login attempts</span></p>
<p class="c1"><span class="c0">Game launch screens</span></p>
<p class="c1"><span class="c0">Online feature access</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Client version outdated</span></p>
<p class="c1"><span class="c0">Compatibility rules enforced</span></p>
<p class="c1"><span class="c0">Service update rollout completed</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Access is blocked</span></p>
<p class="c1"><span class="c0">Update prompt may appear</span></p>
<p class="c1"><span class="c0">Access resumes after update</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a server outage</span></p>
<p class="c1"><span class="c0">It is not a network error</span></p>
<p class="c1"><span class="c0">It is not an account restriction</span></p>
<p class="c1"><span class="c0">Related error codes</span></p>
<p class="c1"><span class="c0">Error Code 800 &mdash; Version mismatch</span></p>
<p class="c1"><span class="c0">Error Code 999 &mdash; Unknown error</span></p>
<p class="c1"><span class="c0">Notes and edge cases</span></p>
<p class="c1"><span class="c0">Update enforcement may roll out gradually. Some regions may receive updates earlier.</span></p>
<p class="c1"><span class="c0">Optional next actions</span></p>
