<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(Error Code 200 &mdash; Connection lost)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that an active connection to the game server was lost.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">Error Code 200 appears when an established connection is interrupted. The issue occurs during an active session. It does not indicate account enforcement. The session ends to maintain consistency. Connectivity or service stability may be involved.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">During gameplay</span></p>
<p class="c1"><span class="c0">Session interruption notices</span></p>
<p class="c1"><span class="c0">Match or lobby disconnections</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Network interruption</span></p>
<p class="c1"><span class="c0">Temporary service instability</span></p>
<p class="c1"><span class="c0">Session timeout</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Session ends</span></p>
<p class="c1"><span class="c0">Player returns to menu</span></p>
<p class="c1"><span class="c0">Reconnection may be possible</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a ban or suspension</span></p>
<p class="c1"><span class="c0">It is not a server shutdown</span></p>
<p class="c1"><span class="c0">It is not a payment issue</span></p>
