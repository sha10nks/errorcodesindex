<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(Error Code 1200 &mdash; Network blocked)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that network access to the service is blocked.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">Error Code 1200 appears when network communication is blocked by a rule or restriction. The block occurs before the request reaches the service. It does not indicate account enforcement. The system cannot establish a connection. The error reflects a connectivity barrier.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">Service connection attempts</span></p>
<p class="c1"><span class="c0">Network access checks</span></p>
<p class="c1"><span class="c0">Online feature entry</span></p>

<p class="c1"><span class="c0">Network access restricted</span></p>
<p class="c1"><span class="c0">Routing or filtering applied</span></p>
<p class="c1"><span class="c0">Connectivity policy enforced</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Connection fails</span></p>
<p class="c1"><span class="c0">Online features unavailable</span></p>
<p class="c1"><span class="c0">Retry may fail until conditions change</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a server outage</span></p>
<p class="c1"><span class="c0">It is not a license issue</span></p>
<p class="c1"><span class="c0">It is not a payment error</span></p>
