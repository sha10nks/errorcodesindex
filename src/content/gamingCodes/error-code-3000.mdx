<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the account is banned from accessing the service.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">Error Code 3000 appears when a platform enforces a ban on an account. The restriction is applied at the account level. It does not represent a system failure. Access is blocked according to policy. The ban may be temporary or permanent.</span></p>
