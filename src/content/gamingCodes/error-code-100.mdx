<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(Error Code 100 &mdash; Server unavailable)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the game server is currently unavailable.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">Error Code 100 appears when a game cannot connect to its server due to unavailability. The issue originates from server infrastructure rather than the client. It does not indicate account enforcement. Multiple users may be affected simultaneously. Access restores once servers recover.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">Game startup screens</span></p>
<p class="c1"><span class="c0">Server connection attempts</span></p>
<p class="c1"><span class="c0">Online mode selection</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Server outage</span></p>
<p class="c1"><span class="c0">Maintenance window</span></p>
<p class="c1"><span class="c0">Infrastructure disruption</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Online gameplay unavailable</span></p>
<p class="c1"><span class="c0">Local features may function</span></p>
<p class="c1"><span class="c0">Access restores later</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a login credential issue</span></p>
<p class="c1"><span class="c0">It is not a network misconfiguration</span></p>
<p class="c1"><span class="c0">It is not a license error</span></p>
