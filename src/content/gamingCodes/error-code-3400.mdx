<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that services are unavailable due to ongoing maintenance.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">Error Code 3400 appears when a platform places systems into maintenance mode. The unavailability is planned or controlled. It does not indicate errors or account problems. Access resumes automatically after maintenance. The restriction is temporary.</span></p>
