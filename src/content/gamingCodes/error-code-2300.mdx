<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that a license is no longer valid.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">Error Code 2300 appears when the platform determines that a license entitlement has expired. The check occurs during access validation. It does not indicate account suspension. Access is blocked until a valid license is detected. The expiration may be time-based or policy-driven.</span></p>
