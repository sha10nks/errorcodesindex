<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(IS-DS01 &mdash; Disk space error)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that insufficient disk space prevented an operation from completing.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">IS-DS01 appears when the launcher detects that available storage is insufficient for an operation. The check occurs before or during installation or updates. It does not indicate a licensing or network issue. The system blocks the operation to avoid partial writes. Storage conditions must change before retrying.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">Game installation attempts</span></p>
<p class="c1"><span class="c0">Update preparation stages</span></p>
<p class="c1"><span class="c0">Storage validation screens</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Insufficient available disk space</span></p>
<p class="c1"><span class="c0">Storage allocation check failed</span></p>
<p class="c1"><span class="c0">Installation size exceeds capacity</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Operation stops</span></p>
<p class="c1"><span class="c0">Content is not installed or updated</span></p>
<p class="c1"><span class="c0">Retry required after storage changes</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a server outage</span></p>
<p class="c1"><span class="c0">It is not a ban or suspension</span></p>
<p class="c1"><span class="c0">It is not a patch validation error</span></p>
<p class="c1"><span class="c0">Related error codes</span></p>
