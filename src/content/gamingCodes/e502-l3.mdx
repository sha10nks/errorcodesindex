<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(E502 L3 &mdash; Transaction failed)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that a Steam transaction failed during processing.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">E502 L3 is returned when Steam cannot process a transaction request. The failure occurs at the service or payment gateway layer. It does not confirm a completed charge. The code reflects a transaction-level interruption rather than a content or account issue. Processing may resume later.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">Checkout process</span></p>
<p class="c1"><span class="c0">Purchase confirmation attempts</span></p>
<p class="c1"><span class="c0">Wallet transaction screens</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Service-side transaction failure</span></p>
<p class="c1"><span class="c0">Payment processing interruption</span></p>
<p class="c1"><span class="c0">Temporary platform instability</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Transaction is not completed</span></p>
<p class="c1"><span class="c0">No content is delivered</span></p>
<p class="c1"><span class="c0">Retry may be required later</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a license error</span></p>
<p class="c1"><span class="c0">It is not a download failure</span></p>
<p class="c1"><span class="c0">It is not an account restriction</span></p>
