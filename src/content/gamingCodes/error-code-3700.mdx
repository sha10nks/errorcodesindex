<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(Error Code 3700 &mdash; Session conflict)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that a session conflict prevented access.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">Error Code 3700 appears when multiple sessions or states conflict. The service blocks access to maintain consistency. It does not indicate enforcement. One session may invalidate another. Resolution occurs when conflicts clear.</span></p>

<p class="c1"><span class="c0">Session ends</span></p>
<p class="c1"><span class="c0">Retry may be required</span></p>
