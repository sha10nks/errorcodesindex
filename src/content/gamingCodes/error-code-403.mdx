<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(Error Code 403 &mdash; Access forbidden)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that access to the requested resource is forbidden.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">Error Code 403 appears when a request is understood but not permitted. The restriction is enforced by the service. It does not indicate a system error. Access may be blocked due to permissions or policy rules. The resource exists but is inaccessible.</span></p>

<p class="c1"><span class="c0">Request completes without content</span></p>
