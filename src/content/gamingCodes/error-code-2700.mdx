<p class="c1"><span class="c0">Title</span></p>

<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the server cannot accept additional connections.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">Error Code 2700 appears when server capacity has been reached. The issue originates from server load management. It does not indicate account enforcement. New connections are blocked until capacity frees up. Access may restore automatically.</span></p>
