<p class="c1"><span class="c0">Title</span></p>

<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that a launcher or game update could not be completed.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">SU-FC02 appears when an update process fails during application or validation. The issue occurs at the update handling stage. It does not indicate licensing or account restrictions. Updates stop to prevent inconsistencies. Retry may be possible later.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">Launcher update processes</span></p>
<p class="c1"><span class="c0">Game update screens</span></p>
<p class="c1"><span class="c0">Error notifications</span></p>

<p class="c1"><span class="c0">Update application failure</span></p>
<p class="c1"><span class="c0">Validation or processing interruption</span></p>
<p class="c1"><span class="c0">Temporary service issue</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Update stops</span></p>
<p class="c1"><span class="c0">Launcher or game remains outdated</span></p>
<p class="c1"><span class="c0">Retry may be required</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a launch failure</span></p>
<p class="c1"><span class="c0">It is not a network ban</span></p>
<p class="c1"><span class="c0">It is not a storage error</span></p>
