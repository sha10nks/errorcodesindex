<p class="c1"><span class="c0">This code indicates that the account has been permanently restricted from PlayStation Network services.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">WS-37368-7 is displayed when PSN applies a permanent enforcement action. The restriction affects the entire account. This is a platform-level decision and is not related to system performance. The code indicates that access to online services is no longer permitted. It does not affect local console functionality.</span></p>
