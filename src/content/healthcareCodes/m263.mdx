<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(M263 &mdash; Coding error)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that an error was identified in coding information.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">M263 is used when CMS identifies incorrect or invalid coding on the claim. Coding accuracy is required for adjudication. This is an administrative issue rather than a coverage determination. Corrected coding may allow reprocessing.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">Medicare remittance advice</span></p>
<p class="c1"><span class="c0">Coding validation reports</span></p>
<p class="c1"><span class="c0">Claim rejection notices</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Incorrect diagnosis or procedure codes</span></p>
<p class="c1"><span class="c0">Coding does not meet CMS rules</span></p>
<p class="c1"><span class="c0">Submission errors</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Claim is denied or adjusted</span></p>
<p class="c1"><span class="c0">Reprocessing may be required</span></p>
<p class="c1"><span class="c0">Payment is delayed</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a non-covered service decision</span></p>
<p class="c1"><span class="c0">It is not patient responsibility</span></p>
<p class="c1"><span class="c0">It is not a contractual adjustment</span></p>
