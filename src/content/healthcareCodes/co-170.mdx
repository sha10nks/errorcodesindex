<p class="c1"><span class="c0">SECTION: Healthcare Insurance / Medical Billing</span></p>
<p class="c1"><span class="c0">SYSTEM: Claim Adjustment Reason Codes (CARC)</span></p>
<p class="c1"><span class="c0">CODE: CO-170</span></p>
<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(CO-170 &mdash; Payment adjusted due to referral absence)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that payment was adjusted because a required referral was not present.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">CO-170 is used when a payer requires a referral for coverage and no valid referral is on file. This is a benefit management requirement tied to plan rules. The code does not reflect service quality or billing accuracy. Coverage eligibility may depend on referral compliance.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">EOB statements</span></p>
<p class="c1"><span class="c0">Remittance advice</span></p>
<p class="c1"><span class="c0">Referral review summaries</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Required referral not obtained</span></p>
<p class="c1"><span class="c0">Referral information missing or invalid</span></p>
<p class="c1"><span class="c0">Referral did not match the service</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Payment is denied or reduced</span></p>
<p class="c1"><span class="c0">Claim processing completes</span></p>
<p class="c1"><span class="c0">Appeal options may exist</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a documentation format error</span></p>
<p class="c1"><span class="c0">It is not a duplicate claim</span></p>
<p class="c1"><span class="c0">It is not a contractual adjustment</span></p>
