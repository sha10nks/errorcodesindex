<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(N95 &mdash; Benefits adjusted)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that benefits were adjusted according to payer rules.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">N95 is used to note that an adjustment was applied to the claim based on benefit calculations. It provides clarification but does not independently determine coverage. Adjustments may relate to coordination of benefits, contractual terms, or cost-sharing. The code does not specify the reason for adjustment on its own.</span></p>

<p class="c1"><span class="c0">Remittance advice remark fields</span></p>
<p class="c1"><span class="c0">EOB summaries</span></p>
<p class="c1"><span class="c0">Claim adjustment details</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Coordination of benefits applied</span></p>
<p class="c1"><span class="c0">Contractual or benefit calculations adjusted</span></p>
<p class="c1"><span class="c0">Cost-sharing rules enforced</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Adjusted payment is finalized</span></p>
<p class="c1"><span class="c0">Claim processing is complete</span></p>
<p class="c1"><span class="c0">Balances are updated accordingly</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a denial</span></p>
<p class="c1"><span class="c0">It is not a billing error</span></p>
<p class="c1"><span class="c0">It is not patient responsibility by itself</span></p>
