<p class="c1"><span class="c0">SECTION: Healthcare Insurance / Medical Billing</span></p>
<p class="c1"><span class="c0">SYSTEM: Patient Responsibility (PR)</span></p>
<p class="c1"><span class="c0">CODE: PR-2</span></p>
<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(PR-2 &mdash; Coinsurance)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the patient is responsible for a percentage of the covered charges.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">PR-2 reflects the coinsurance portion assigned to the patient after insurance payment. Coinsurance represents a shared cost arrangement between the insurer and patient. This is a normal benefit structure, not a denial. The service remains covered, with costs divided according to plan terms.</span></p>

<p class="c1"><span class="c0">Remittance advice</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Coinsurance applies after deductible</span></p>
<p class="c1"><span class="c0">Plan cost-sharing requirements</span></p>
<p class="c1"><span class="c0">Coverage percentage rules enforced</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Patient balance includes coinsurance amount</span></p>
<p class="c1"><span class="c0">Claim processing is complete</span></p>
<p class="c1"><span class="c0">Remaining charges are settled</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a non-covered charge</span></p>
<p class="c1"><span class="c0">It is not a billing error</span></p>
<p class="c1"><span class="c0">It is not a contractual adjustment</span></p>
