<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(N272 &mdash; Coordination of benefits issue)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that a coordination of benefits issue affected claim processing.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">N272 is used when multiple coverages exist and payer responsibility needs clarification. Coordination of benefits determines which payer is primary. The code does not deny the service itself. Payment outcomes depend on payer order and policy rules.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">Remittance advice remarks</span></p>
<p class="c1"><span class="c0">EOB statements</span></p>
<p class="c1"><span class="c0">Coordination of benefits notices</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Multiple insurance policies exist</span></p>
<p class="c1"><span class="c0">Primary payer not clearly identified</span></p>
<p class="c1"><span class="c0">Prior payer adjudication required</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Claim processing is adjusted or delayed</span></p>
<p class="c1"><span class="c0">Responsibility may shift between payers</span></p>
<p class="c1"><span class="c0">Final payment issued after coordination</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a coverage exclusion</span></p>
<p class="c1"><span class="c0">It is not a billing error</span></p>
<p class="c1"><span class="c0">It is not patient responsibility by default</span></p>
<p class="c1"><span class="c0">Related error codes</span></p>

<p class="c1"><span class="c0">Notes and edge cases</span></p>
<p class="c1"><span class="c0">Coordination rules vary by payer. Timing of adjudication exchanges can affect outcomes.</span></p>
