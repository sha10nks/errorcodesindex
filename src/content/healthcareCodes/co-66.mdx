<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(CO-66 &mdash; Incomplete procedure code)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the procedure code submitted was incomplete or improperly formatted.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">CO-66 is used when a procedure code lacks required components. Complete codes are necessary for payer adjudication. This is a formatting and completeness issue rather than a coverage determination. The service may be eligible once a complete code is submitted.</span></p>

<p class="c1"><span class="c0">Claim is rejected or denied</span></p>
<p class="c1"><span class="c0">Reprocessing may be required</span></p>
<p class="c1"><span class="c0">Payment is delayed</span></p>
