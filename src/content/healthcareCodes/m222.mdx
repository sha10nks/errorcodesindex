<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(M222 &mdash; Authorization issue)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that an authorization-related issue affected claim processing.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">M222 is used when CMS identifies a problem related to authorization requirements. This may involve missing, invalid, or unmatched authorization records. The code reflects benefit management rules rather than service quality. Coverage eligibility depends on meeting authorization criteria.</span></p>

<p class="c1"><span class="c0">Authorization not obtained</span></p>
<p class="c1"><span class="c0">Authorization details missing or incorrect</span></p>
<p class="c1"><span class="c0">Authorization expired</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Payment is denied or delayed</span></p>
