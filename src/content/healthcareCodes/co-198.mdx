<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(CO-198 &mdash; Preauthorization missing)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that required preauthorization was not obtained before services were provided.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">CO-198 is applied when payer rules require preauthorization and none is on file. This reflects benefit management policies rather than service quality. Coverage eligibility depends on meeting authorization requirements. The code does not indicate improper care delivery.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">EOB statements</span></p>
<p class="c1"><span class="c0">Remittance advice</span></p>
<p class="c1"><span class="c0">Authorization review notices</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Preauthorization not requested</span></p>
<p class="c1"><span class="c0">Authorization details missing or invalid</span></p>
<p class="c1"><span class="c0">Authorization expired</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Payment is denied or adjusted</span></p>
<p class="c1"><span class="c0">Claim processing completes</span></p>
<p class="c1"><span class="c0">Appeal options may exist</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a billing format error</span></p>
<p class="c1"><span class="c0">It is not patient responsibility</span></p>
<p class="c1"><span class="c0">It is not a contractual adjustment</span></p>
