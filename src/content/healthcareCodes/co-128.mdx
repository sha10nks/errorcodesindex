<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(CO-128 &mdash; Invalid billing modifier)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the modifier submitted is invalid or not allowed for the billed service.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">CO-128 is applied when a modifier does not meet payer rules for the reported procedure. Modifiers provide context for how services are performed. Invalid modifiers prevent accurate adjudication. This is a data validation issue rather than a coverage determination.</span></p>
