<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(CO-65 &mdash; Procedure code invalid)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the procedure code submitted is not valid according to payer rules.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">CO-65 is applied when a procedure code does not meet payer validation requirements. This may relate to outdated, incorrect, or unsupported codes. The issue is administrative rather than a coverage decision. Valid procedure codes are required for accurate adjudication.</span></p>

<p class="c1"><span class="c0">Claim rejection notices</span></p>
<p class="c1"><span class="c0">Remittance advice</span></p>
<p class="c1"><span class="c0">Billing validation reports</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Invalid or discontinued procedure code used</span></p>
<p class="c1"><span class="c0">Code does not align with payer system rules</span></p>
<p class="c1"><span class="c0">Submission format errors</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Claim processing stops</span></p>
<p class="c1"><span class="c0">Reprocessing may be required</span></p>
<p class="c1"><span class="c0">Payment is delayed</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a non-covered service decision</span></p>
<p class="c1"><span class="c0">It is not patient responsibility</span></p>
<p class="c1"><span class="c0">It is not a contractual adjustment</span></p>
