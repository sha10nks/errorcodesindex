<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(N503 &mdash; Incorrect billing)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that billing information was incorrect according to payer rules.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">N503 is used when billing data does not conform to payer requirements. This is a general billing validation remark. It does not specify the exact issue and typically supports other adjustment codes. The code does not imply coverage denial or service issues.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">Remittance advice remarks</span></p>
<p class="c1"><span class="c0">Claim adjustment notices</span></p>
<p class="c1"><span class="c0">Billing error summaries</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Incorrect billing details submitted</span></p>
<p class="c1"><span class="c0">Coding or formatting errors</span></p>
<p class="c1"><span class="c0">Payer billing rules enforced</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Claim is denied or adjusted</span></p>
<p class="c1"><span class="c0">Reprocessing may be required</span></p>
<p class="c1"><span class="c0">Payment is delayed</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a coverage exclusion</span></p>
<p class="c1"><span class="c0">It is not patient responsibility</span></p>
<p class="c1"><span class="c0">It is not a contractual adjustment</span></p>
