<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(M15 &mdash; Separately billed services)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that services were billed separately when they are typically included in another service.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">M15 is used when CMS determines that certain services should not be billed separately because they are considered part of another service. This reflects Medicare billing policy rather than coverage eligibility. The code does not imply improper service delivery. It identifies bundling rules applied during claim adjudication.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">Medicare remittance advice</span></p>
<p class="c1"><span class="c0">CMS claim adjustment notices</span></p>
<p class="c1"><span class="c0">EOB statements for Medicare claims</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Services bundled under Medicare rules</span></p>
<p class="c1"><span class="c0">Separate billing not allowed</span></p>
<p class="c1"><span class="c0">Medicare payment policies enforced</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Payment is reduced or adjusted</span></p>
<p class="c1"><span class="c0">Bundled service remains reimbursed</span></p>
<p class="c1"><span class="c0">Claim processing completes</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a non-covered service</span></p>
<p class="c1"><span class="c0">It is not a documentation error</span></p>
<p class="c1"><span class="c0">It is not patient responsibility</span></p>

<p class="c1"><span class="c0">Bundling rules may change with policy updates. Some services are separately payable under limited circumstances.</span></p>
