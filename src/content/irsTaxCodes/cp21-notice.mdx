<p class="c1"><span class="c0">SECTION: IRS / Tax Status &amp; Refund Codes</span></p>

<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(CP21 notice &mdash; Amended return adjustment notice)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This notice indicates that the IRS processed an amended return and adjusted the account accordingly.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">A CP21 notice reflects that changes from an amended return were applied to the account. It explains adjustments made and their effect on refund or balance. The notice signals completion of amendment processing. It does not indicate audit activity by itself. Transcript entries typically show related assessment or reduction codes.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">In mailed IRS correspondence</span></p>
<p class="c1"><span class="c0">On account transcripts following code 977</span></p>
<p class="c1"><span class="c0">During amended return processing</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Amended return was submitted</span></p>
<p class="c1"><span class="c0">IRS completed review of revised figures</span></p>
<p class="c1"><span class="c0">Account adjustments were required</span></p>
<p class="c1"><span class="c0">Correction of reported amounts occurred</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Balance may increase or decrease</span></p>
<p class="c1"><span class="c0">Refund may be issued if overpayment exists</span></p>
<p class="c1"><span class="c0">Additional adjustment entries may post</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a freeze</span></p>
<p class="c1"><span class="c0">It is not an audit notice</span></p>
<p class="c1"><span class="c0">It does not indicate fraud</span></p>
