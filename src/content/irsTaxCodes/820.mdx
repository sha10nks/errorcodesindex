<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(820 &mdash; Credit offset)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that a credit was applied to an outstanding balance or obligation.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">Code 820 reflects that a credit was used to offset an existing liability. This entry represents application of available funds toward outstanding tax or other obligations. It does not indicate refund issuance because the credit was applied internally. The code reflects reduction of one balance using available credit. It signals administrative application of funds within IRS systems.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">On IRS account transcripts</span></p>
<p class="c1"><span class="c0">After credits are applied to balances</span></p>
<p class="c1"><span class="c0">During review of account adjustments</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Overpayment existed and was applied</span></p>
<p class="c1"><span class="c0">Outstanding tax liability required offset</span></p>
<p class="c1"><span class="c0">Administrative reconciliation occurred</span></p>
<p class="c1"><span class="c0">Cross-year balances were adjusted</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">The outstanding balance decreases</span></p>
<p class="c1"><span class="c0">Credit balances may reduce</span></p>
<p class="c1"><span class="c0">Additional adjustment entries may appear</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a refund issuance</span></p>
<p class="c1"><span class="c0">It is not a freeze or hold</span></p>
<p class="c1"><span class="c0">It does not represent new assessment</span></p>
