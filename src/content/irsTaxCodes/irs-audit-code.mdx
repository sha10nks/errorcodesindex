<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(IRS audit code &mdash; Examination initiated indicator)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This status indicates that the IRS has initiated an examination or audit review process.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">&ldquo;IRS audit code&rdquo; generally refers to transcript entries such as code 420 that indicate examination initiation. It reflects that the account has been selected for review beyond routine processing. The code does not specify the scope or outcome of the examination. It signals that review procedures are active within IRS systems. The transcript provides additional context regarding related adjustments.</span></p>
