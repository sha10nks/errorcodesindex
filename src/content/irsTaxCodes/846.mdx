<p class="c1"><span class="c0">SECTION: IRS / Tax Status &amp; Refund Codes</span></p>
<p class="c1"><span class="c0">SYSTEM: IRS Account Transcript / Refund Processing</span></p>
<p class="c1"><span class="c0">CODE: 846</span></p>
<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(846 &mdash; Refund issued)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the IRS has issued a refund for the specified tax period.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">Code 846 reflects that a refund has been authorized and issued. The entry typically includes the refund amount and an issue date. This code represents a completed refund transaction in IRS records. It indicates that the refund has moved from processing to issuance status. The presence of code 846 does not specify the method of payment within the code itself, but it confirms that issuance has occurred at the system level.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">On IRS account transcripts</span></p>
<p class="c1"><span class="c0">After processing and release of a refund</span></p>
<p class="c1"><span class="c0">When reviewing refund-related transaction entries</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Refund processing completed successfully</span></p>
<p class="c1"><span class="c0">Holds or pending actions were resolved</span></p>
<p class="c1"><span class="c0">Credits exceeded tax liability</span></p>
<p class="c1"><span class="c0">The IRS finalized payment authorization</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">The refund is transmitted according to the payment method on record</span></p>
<p class="c1"><span class="c0">Account balance updates reflect the refund issuance</span></p>
<p class="c1"><span class="c0">No further refund-related codes may appear unless adjustments occur</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a pending status</span></p>
<p class="c1"><span class="c0">It is not a hold or freeze</span></p>
<p class="c1"><span class="c0">It does not indicate examination by itself</span></p>

<p class="c1"><span class="c0">Notes and edge cases</span></p>
<p class="c1"><span class="c0">The issue date associated with code 846 reflects the IRS system date of issuance. In some cases, additional codes may appear after 846 if offsets or adjustments are applied. The transcript serves as official confirmation of issuance at the account level.</span></p>
<p class="c1"><span class="c0">Optional next actions</span></p>
<p class="c1"><span class="c0">Review the issue date and refund amount</span></p>
<p class="c1"><span class="c0">Monitor financial account records for deposit confirmation</span></p>
<p class="c1"><span class="c0">Check for any subsequent adjustment entries</span></p>
<p class="c1"><span class="c0">Consult official IRS resources if discrepancies arise</span></p>
