<p class="c1"><span class="c0">CODE: ER DH</span></p>
<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(ER DH &mdash; Defrost heater error)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the refrigerator detected an irregularity in the defrost heating system.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">ER DH appears when temperature sensors do not detect expected heating during the defrost cycle. The appliance monitors temperature changes during automatic defrost operations. If heating thresholds are not achieved, the system records this code. This reflects a defrost monitoring response. It does not automatically confirm heater failure.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">On the refrigerator display</span></p>
<p class="c1"><span class="c0">During defrost cycle</span></p>
<p class="c1"><span class="c0">After power interruption</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Temperature increase not detected</span></p>
<p class="c1"><span class="c0">Defrost cycle exceeded programmed duration</span></p>
<p class="c1"><span class="c0">Sensor reading outside range</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Defrost process may pause</span></p>
<p class="c1"><span class="c0">Cooling may continue</span></p>
<p class="c1"><span class="c0">Code remains displayed</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a compressor failure</span></p>
<p class="c1"><span class="c0">It is not a communication error</span></p>
<p class="c1"><span class="c0">It is not a leak alert</span></p>
<p class="c1"><span class="c0">Related error codes</span></p>
<p class="c1"><span class="c0">E09 &mdash; Heating circuit failure</span></p>
<p class="c1"><span class="c0">ER FF &mdash; Freezer fan error</span></p>
<p class="c1"><span class="c0">ER CO &mdash; Cooling error</span></p>
<p class="c1"><span class="c0">Notes and edge cases</span></p>
<p class="c1"><span class="c0">Heavy frost buildup may influence defrost sensor readings.</span></p>
<p class="c1"><span class="c0">Optional next actions</span></p>
<p class="c1"><span class="c0">Allow appliance to complete cycle</span></p>
<p class="c1"><span class="c0">Monitor temperature stability</span></p>
<p class="c1"><span class="c0">Observe recurrence</span></p>
<p class="c1"><span class="c0">External help (optional)</span></p>
<p class="c1"><span class="c0">Persistent defrost alerts may require professional evaluation.</span></p>
