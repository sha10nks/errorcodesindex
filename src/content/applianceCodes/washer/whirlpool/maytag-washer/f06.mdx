<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(F06 &mdash; Motor control error)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the washer detected an issue with motor communication or operation.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">F06 appears when the appliance control board cannot properly regulate or receive expected signals from the motor system. The washer continuously monitors rotation feedback. If irregular or absent signals are detected, the cycle halts. This code reflects a control communication condition. It does not automatically confirm motor failure.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">During wash agitation</span></p>
<p class="c1"><span class="c0">During spin cycle</span></p>
<p class="c1"><span class="c0">On the control display</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Motor signal mismatch</span></p>
<p class="c1"><span class="c0">Rotation feedback interruption</span></p>
<p class="c1"><span class="c0">Control communication timeout</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Drum stops moving</span></p>
<p class="c1"><span class="c0">Cycle pauses</span></p>
<p class="c1"><span class="c0">Machine remains powered on</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a drain issue</span></p>
<p class="c1"><span class="c0">It is not a fill error</span></p>
<p class="c1"><span class="c0">It is not a heating malfunction</span></p>
<p class="c1"><span class="c0">Related error codes</span></p>
<p class="c1"><span class="c0">F21 &mdash; Long drain</span></p>
<p class="c1"><span class="c0">LE &mdash; Motor locked</span></p>
<p class="c1"><span class="c0">F01 &mdash; Control board failure</span></p>
<p class="c1"><span class="c0">Notes and edge cases</span></p>
<p class="c1"><span class="c0">Temporary communication disruptions may cause intermittent alerts.</span></p>
<p class="c1"><span class="c0">Optional next actions</span></p>
<p class="c1"><span class="c0">Restart the cycle</span></p>
<p class="c1"><span class="c0">Observe drum behavior</span></p>
<p class="c1"><span class="c0">Monitor for recurrence</span></p>
