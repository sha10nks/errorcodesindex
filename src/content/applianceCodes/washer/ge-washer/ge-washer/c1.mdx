<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(C1 &mdash; Water level issue)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the washer detected irregular water level readings.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">C1 appears when internal sensors report inconsistent water level changes. The appliance continuously compares expected level changes with actual readings. If discrepancies persist, the system pauses operation. This code reflects a monitoring safeguard. It does not confirm sensor failure.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">During fill stage</span></p>
<p class="c1"><span class="c0">Mid-cycle</span></p>
<p class="c1"><span class="c0">On the control panel</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Water level threshold not met</span></p>
<p class="c1"><span class="c0">Sensor readings inconsistent</span></p>
<p class="c1"><span class="c0">Fill duration exceeded</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Cycle pauses</span></p>
<p class="c1"><span class="c0">Fill stops</span></p>
<p class="c1"><span class="c0">Appliance awaits reset</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a drain blockage</span></p>
<p class="c1"><span class="c0">It is not a heating fault</span></p>
<p class="c1"><span class="c0">It is not a door lock error</span></p>
