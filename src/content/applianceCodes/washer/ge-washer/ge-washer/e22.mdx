<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(E22 &mdash; Drain problem)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the appliance detected a drain timing issue.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">E22 appears when the washer does not complete the drain phase within programmed limits. The system monitors water level changes during draining. If expected reduction does not occur, the cycle pauses. This code reflects drain monitoring logic. It does not confirm pump damage.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">Toward end of wash cycle</span></p>
<p class="c1"><span class="c0">Before spin stage</span></p>
<p class="c1"><span class="c0">On display panel</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Drain duration exceeded threshold</span></p>
<p class="c1"><span class="c0">Water level unchanged</span></p>
<p class="c1"><span class="c0">Protective timeout activated</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Cycle pauses</span></p>
<p class="c1"><span class="c0">Spin does not start</span></p>
<p class="c1"><span class="c0">Water remains inside drum</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a heating issue</span></p>
<p class="c1"><span class="c0">It is not a fill error</span></p>
<p class="c1"><span class="c0">It is not a door lock problem</span></p>
<p class="c1"><span class="c0">Related error codes</span></p>
<p class="c1"><span class="c0">E24 &mdash; Drain blockage</span></p>
<p class="c1"><span class="c0">5E &mdash; Drain error</span></p>
<p class="c1"><span class="c0">F21 &mdash; Long drain</span></p>
<p class="c1"><span class="c0">Notes and edge cases</span></p>
<p class="c1"><span class="c0">Heavy loads may increase drain time slightly.</span></p>
<p class="c1"><span class="c0">Optional next actions</span></p>
<p class="c1"><span class="c0">Restart cycle</span></p>
<p class="c1"><span class="c0">Observe drum water level</span></p>
<p class="c1"><span class="c0">Monitor for recurrence</span></p>
<p class="c1"><span class="c0">External help (optional)</span></p>
<p class="c1"><span class="c0">Ongoing drain alerts may require inspection.</span></p>
