<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(F21 &mdash; Motor fault)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the washer detected irregular motor performance.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">F21 appears when the appliance senses inconsistent or absent motor rotation signals. The system monitors rotational feedback during agitation and spin. If irregularities exceed thresholds, the cycle stops. This reflects motor protection logic. It does not automatically confirm permanent motor failure.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">During wash agitation</span></p>
<p class="c1"><span class="c0">During spin cycle</span></p>
<p class="c1"><span class="c0">On control display</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Rotation signal mismatch</span></p>
<p class="c1"><span class="c0">Motor response delay</span></p>
<p class="c1"><span class="c0">Feedback threshold exceeded</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Drum stops</span></p>
<p class="c1"><span class="c0">Cycle pauses</span></p>
<p class="c1"><span class="c0">Appliance remains powered</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a drain issue</span></p>
<p class="c1"><span class="c0">It is not a heating malfunction</span></p>
<p class="c1"><span class="c0">It is not a fill problem</span></p>
<p class="c1"><span class="c0">Related error codes</span></p>
<p class="c1"><span class="c0">F06 &mdash; Motor control error</span></p>
<p class="c1"><span class="c0">LE &mdash; Motor locked</span></p>
<p class="c1"><span class="c0">E3 &mdash; Load or balance error</span></p>
<p class="c1"><span class="c0">Notes and edge cases</span></p>
<p class="c1"><span class="c0">Heavy or uneven loads may influence motor feedback readings.</span></p>
<p class="c1"><span class="c0">Optional next actions</span></p>
<p class="c1"><span class="c0">Restart cycle</span></p>
<p class="c1"><span class="c0">Redistribute load</span></p>
<p class="c1"><span class="c0">Monitor recurrence</span></p>
<p class="c1"><span class="c0">External help (optional)</span></p>
<p class="c1"><span class="c0">Ongoing motor alerts may require service evaluation.</span></p>
