<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(E24 &mdash; Drain blockage)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the washer detected a blockage or restricted drainage condition.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">E24 appears when the system senses that drainage is not occurring at normal speed. Internal monitoring compares expected drain performance against actual results. If discrepancy exceeds limits, the cycle pauses. This code reflects a flow monitoring response. It does not automatically confirm mechanical damage.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">During drain phase</span></p>
<p class="c1"><span class="c0">Before spin</span></p>
<p class="c1"><span class="c0">On the control display</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Drain flow below expected rate</span></p>
<p class="c1"><span class="c0">Water level remains high</span></p>
<p class="c1"><span class="c0">Protective drain timeout triggered</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Cycle pauses</span></p>
<p class="c1"><span class="c0">Water may remain inside drum</span></p>
<p class="c1"><span class="c0">Appliance awaits reset</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a fill issue</span></p>
<p class="c1"><span class="c0">It is not a heating fault</span></p>
<p class="c1"><span class="c0">It is not a motor imbalance</span></p>
<p class="c1"><span class="c0">Related error codes</span></p>
<p class="c1"><span class="c0">E22 &mdash; Drain problem</span></p>
<p class="c1"><span class="c0">OE &mdash; Drain error</span></p>
<p class="c1"><span class="c0">E25 &mdash; Drain pump blocked</span></p>
<p class="c1"><span class="c0">Notes and edge cases</span></p>
<p class="c1"><span class="c0">Temporary obstructions can cause intermittent alerts.</span></p>
<p class="c1"><span class="c0">Optional next actions</span></p>
<p class="c1"><span class="c0">Restart after reset</span></p>
<p class="c1"><span class="c0">Monitor water removal</span></p>
<p class="c1"><span class="c0">Observe if code returns</span></p>
<p class="c1"><span class="c0">External help (optional)</span></p>
<p class="c1"><span class="c0">Persistent drain alerts may require professional assessment.</span></p>
