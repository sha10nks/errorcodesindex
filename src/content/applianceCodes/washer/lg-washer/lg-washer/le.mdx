<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(LE &mdash; Motor locked)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the washer detected restricted motor movement.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">LE appears when the motor system detects abnormal resistance or lack of rotation. The appliance monitors rotational feedback signals. If expected movement does not occur, the cycle stops. This code reflects a motor protection response. It does not automatically confirm permanent motor damage.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">During wash agitation</span></p>
<p class="c1"><span class="c0">During spin cycle</span></p>
<p class="c1"><span class="c0">On control display</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Motor rotation signal interrupted</span></p>
<p class="c1"><span class="c0">Overload condition detected</span></p>
<p class="c1"><span class="c0">Feedback inconsistency</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Drum stops moving</span></p>
<p class="c1"><span class="c0">Cycle pauses</span></p>
<p class="c1"><span class="c0">System may attempt restart</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a water supply issue</span></p>
<p class="c1"><span class="c0">It is not a heating problem</span></p>
<p class="c1"><span class="c0">It is not a leak alert</span></p>
<p class="c1"><span class="c0">Related error codes</span></p>
<p class="c1"><span class="c0">F06 &mdash; Motor control error</span></p>
<p class="c1"><span class="c0">UE &mdash; Unbalanced load</span></p>
<p class="c1"><span class="c0">E3 &mdash; Load or balance error</span></p>
<p class="c1"><span class="c0">Notes and edge cases</span></p>
<p class="c1"><span class="c0">Large or uneven loads may trigger temporary motor alerts.</span></p>
<p class="c1"><span class="c0">Optional next actions</span></p>
<p class="c1"><span class="c0">Restart cycle</span></p>
<p class="c1"><span class="c0">Redistribute load</span></p>
<p class="c1"><span class="c0">Observe recurrence</span></p>
<p class="c1"><span class="c0">External help (optional)</span></p>
<p class="c1"><span class="c0">Ongoing motor alerts may require service inspection.</span></p>
