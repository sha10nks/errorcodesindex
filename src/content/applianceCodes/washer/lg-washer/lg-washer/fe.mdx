<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(FE &mdash; Overflow error)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the washer detected excessive water levels.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">FE appears when internal sensors detect water levels exceeding safe thresholds. The appliance monitors fill levels continuously. If levels exceed programmed limits, draining may activate automatically. This code reflects a safety overflow response. It does not confirm permanent system damage.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">During fill stage</span></p>
<p class="c1"><span class="c0">Mid-cycle</span></p>
<p class="c1"><span class="c0">On control display</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Water level threshold exceeded</span></p>
<p class="c1"><span class="c0">Sensor detected excessive fill</span></p>
<p class="c1"><span class="c0">Protective drain triggered</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Fill stops</span></p>
<p class="c1"><span class="c0">Drain may activate</span></p>
<p class="c1"><span class="c0">Cycle pauses</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a drain blockage</span></p>
<p class="c1"><span class="c0">It is not a motor fault</span></p>
<p class="c1"><span class="c0">It is not a heating error</span></p>
<p class="c1"><span class="c0">Related error codes</span></p>
<p class="c1"><span class="c0">E4 &mdash; Overflow condition</span></p>
<p class="c1"><span class="c0">IE &mdash; Water inlet error</span></p>
<p class="c1"><span class="c0">C1 &mdash; Water level issue</span></p>
<p class="c1"><span class="c0">Notes and edge cases</span></p>
<p class="c1"><span class="c0">Foam or suds buildup may affect sensor readings.</span></p>
<p class="c1"><span class="c0">Optional next actions</span></p>
<p class="c1"><span class="c0">Allow system to drain</span></p>
<p class="c1"><span class="c0">Restart cycle after reset</span></p>
