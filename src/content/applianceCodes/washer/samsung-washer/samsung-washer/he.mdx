<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(HE &mdash; Heating error)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the appliance detected a heating system irregularity.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">HE appears when the washer detects that water temperature is not increasing as expected. Internal sensors monitor heating progression. If temperature change does not match programmed parameters, the cycle stops. This code reflects a heating control monitoring response. It does not automatically confirm heating element failure.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">During wash phase</span></p>
<p class="c1"><span class="c0">Mid-cycle</span></p>
<p class="c1"><span class="c0">On the display panel</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Temperature increase not detected</span></p>
<p class="c1"><span class="c0">Heating time exceeded limit</span></p>
<p class="c1"><span class="c0">Sensor threshold not met</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Cycle pauses</span></p>
<p class="c1"><span class="c0">Heating function stops</span></p>
<p class="c1"><span class="c0">Wash may not complete</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a drain problem</span></p>
<p class="c1"><span class="c0">It is not a water inlet issue</span></p>
<p class="c1"><span class="c0">It is not a balance error</span></p>
<p class="c1"><span class="c0">Related error codes</span></p>
<p class="c1"><span class="c0">E2 &mdash; Heating issue</span></p>
<p class="c1"><span class="c0">E09 &mdash; Heating circuit failure</span></p>
<p class="c1"><span class="c0">tE &mdash; Temperature sensor error</span></p>
<p class="c1"><span class="c0">Notes and edge cases</span></p>
<p class="c1"><span class="c0">Cold water cycles may affect sensor behavior if incorrectly configured.</span></p>
