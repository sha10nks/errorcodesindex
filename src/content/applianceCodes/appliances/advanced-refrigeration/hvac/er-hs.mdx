<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(ER HS &mdash; Humidity sensor issue)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the appliance detected irregular humidity sensor readings.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">ER HS appears when the system detects humidity levels outside expected sensor parameters. Advanced refrigeration systems monitor humidity to regulate defrost and airflow. If readings are inconsistent or absent, the appliance records this code. This reflects environmental monitoring logic. It does not automatically confirm cooling failure.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">On control display</span></p>
<p class="c1"><span class="c0">During humidity regulation</span></p>
<p class="c1"><span class="c0">After temperature change</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Sensor reading outside range</span></p>
<p class="c1"><span class="c0">Calibration mismatch</span></p>
<p class="c1"><span class="c0">Monitoring threshold exceeded</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Certain humidity functions may pause</span></p>
<p class="c1"><span class="c0">Display shows code</span></p>
<p class="c1"><span class="c0">Cooling may continue</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a compressor malfunction</span></p>
<p class="c1"><span class="c0">It is not a defrost heater error</span></p>
<p class="c1"><span class="c0">It is not a voltage issue</span></p>
<p class="c1"><span class="c0">Related error codes</span></p>
<p class="c1"><span class="c0">E7 &mdash; Sensor calibration issue</span></p>
<p class="c1"><span class="c0">ER CO &mdash; Cooling error</span></p>
<p class="c1"><span class="c0">F3 &mdash; Temperature sensor fault</span></p>
<p class="c1"><span class="c0">Notes and edge cases</span></p>
<p class="c1"><span class="c0">Rapid environmental changes may temporarily affect sensor readings.</span></p>
<p class="c1"><span class="c0">Optional next actions</span></p>
<p class="c1"><span class="c0">Restart system</span></p>
<p class="c1"><span class="c0">Monitor humidity levels</span></p>
<p class="c1"><span class="c0">Observe recurrence</span></p>
<p class="c1"><span class="c0">External help (optional)</span></p>
<p class="c1"><span class="c0">Repeated humidity alerts may require service evaluation.</span></p>
