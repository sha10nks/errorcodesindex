<p class="c1"><span class="c0">CODE: ER CO</span></p>
<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(ER CO &mdash; Cooling error)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the appliance detected irregular cooling performance.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">ER CO appears when internal temperature monitoring does not reflect expected cooling progression. Refrigeration and HVAC systems compare real-time temperature readings with programmed targets. If cooling thresholds are not achieved within defined timeframes, the system records this code. This reflects performance monitoring logic. It does not automatically confirm compressor failure.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">On display panel</span></p>
<p class="c1"><span class="c0">During active cooling</span></p>
<p class="c1"><span class="c0">After temperature fluctuation</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Cooling rate below expected level</span></p>
<p class="c1"><span class="c0">Temperature not decreasing as programmed</span></p>
<p class="c1"><span class="c0">Monitoring threshold exceeded</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Cooling performance may be reduced</span></p>
<p class="c1"><span class="c0">Alarm may sound</span></p>
<p class="c1"><span class="c0">Code remains displayed</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a defrost heater error</span></p>
<p class="c1"><span class="c0">It is not a communication issue</span></p>
<p class="c1"><span class="c0">It is not a door latch malfunction</span></p>
<p class="c1"><span class="c0">Related error codes</span></p>
<p class="c1"><span class="c0">ER FF &mdash; Freezer fan error</span></p>
<p class="c1"><span class="c0">E6 &mdash; Communication fault</span></p>
<p class="c1"><span class="c0">ER HS &mdash; Humidity sensor issue</span></p>
<p class="c1"><span class="c0">Notes and edge cases</span></p>
<p class="c1"><span class="c0">High ambient temperatures may influence cooling rate readings.</span></p>
<p class="c1"><span class="c0">Optional next actions</span></p>
<p class="c1"><span class="c0">Monitor temperature stability</span></p>
