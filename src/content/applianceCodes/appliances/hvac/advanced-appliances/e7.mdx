<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(E7 &mdash; Sensor calibration issue)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the appliance detected an irregularity in sensor calibration or reference values.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">E7 appears when the system identifies that a sensor&rsquo;s readings fall outside expected calibration parameters. HVAC and advanced appliances use calibrated sensors to regulate temperature, airflow, or humidity. If internal comparisons detect inconsistencies, the cycle may pause. This code reflects monitoring safeguards related to measurement accuracy. It does not automatically confirm complete sensor failure.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">During operation</span></p>
<p class="c1"><span class="c0">After startup</span></p>
<p class="c1"><span class="c0">On the display panel</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Sensor readings inconsistent</span></p>
<p class="c1"><span class="c0">Calibration threshold exceeded</span></p>
<p class="c1"><span class="c0">Internal validation mismatch</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Operation may pause</span></p>
<p class="c1"><span class="c0">System limits certain functions</span></p>
<p class="c1"><span class="c0">Code remains visible</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not necessarily a heating failure</span></p>
<p class="c1"><span class="c0">It is not a voltage issue</span></p>
<p class="c1"><span class="c0">It is not a communication error</span></p>
<p class="c1"><span class="c0">Related error codes</span></p>
<p class="c1"><span class="c0">F3 &mdash; Temperature sensor fault</span></p>
<p class="c1"><span class="c0">ER HS &mdash; Humidity sensor issue</span></p>
<p class="c1"><span class="c0">E1 &mdash; Sensor failure</span></p>
<p class="c1"><span class="c0">Notes and edge cases</span></p>
<p class="c1"><span class="c0">Environmental changes may temporarily affect sensor accuracy readings.</span></p>
<p class="c1"><span class="c0">Optional next actions</span></p>

<p class="c1"><span class="c0">Monitor sensor performance</span></p>
