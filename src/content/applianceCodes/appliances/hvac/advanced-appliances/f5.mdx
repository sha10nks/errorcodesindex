<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(F5 &mdash; Door latch problem)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the appliance detected an issue with the door latch confirmation system.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">F5 appears when the appliance cannot confirm secure door engagement. Advanced appliances include safety interlocks that prevent operation if the door is not properly latched. If confirmation signals are inconsistent or absent, the system halts. This code reflects a safety monitoring response. It does not automatically indicate mechanical damage.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">At startup</span></p>
<p class="c1"><span class="c0">When attempting to begin a cycle</span></p>
<p class="c1"><span class="c0">On the display panel</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Latch confirmation signal missing</span></p>
<p class="c1"><span class="c0">Door alignment irregular</span></p>
<p class="c1"><span class="c0">Safety timeout triggered</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Cycle does not begin</span></p>
<p class="c1"><span class="c0">Appliance remains idle</span></p>
<p class="c1"><span class="c0">Code stays visible</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a control board failure</span></p>
<p class="c1"><span class="c0">It is not a temperature issue</span></p>
<p class="c1"><span class="c0">It is not a voltage irregularity</span></p>
<p class="c1"><span class="c0">Related error codes</span></p>
<p class="c1"><span class="c0">dE &mdash; Door error</span></p>
<p class="c1"><span class="c0">F22 &mdash; Door lock error</span></p>
<p class="c1"><span class="c0">LOC &mdash; Panel locked</span></p>
<p class="c1"><span class="c0">Notes and edge cases</span></p>
<p class="c1"><span class="c0">Minor alignment shifts may cause intermittent latch detection alerts.</span></p>
<p class="c1"><span class="c0">Optional next actions</span></p>
<p class="c1"><span class="c0">Ensure door is fully closed</span></p>
<p class="c1"><span class="c0">Restart cycle</span></p>
<p class="c1"><span class="c0">Monitor recurrence</span></p>
<p class="c1"><span class="c0">External help (optional)</span></p>
<p class="c1"><span class="c0">If latch alerts persist, service inspection may be required.</span></p>
