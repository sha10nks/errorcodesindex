<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(UO &mdash; Vent blockage)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the appliance detected restricted airflow or vent obstruction.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">UO appears when airflow sensors detect insufficient air movement through the venting system. Appliances such as dryers and HVAC units monitor airflow for safety and efficiency. If airflow drops below safe thresholds, the system pauses. This code reflects airflow protection logic. It does not automatically confirm mechanical failure.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">During drying cycle</span></p>
<p class="c1"><span class="c0">Mid-operation</span></p>
<p class="c1"><span class="c0">On the control display</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Airflow below expected level</span></p>
<p class="c1"><span class="c0">Vent monitoring threshold exceeded</span></p>
<p class="c1"><span class="c0">Safety restriction detected</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Heating may pause</span></p>
<p class="c1"><span class="c0">Cycle may stop</span></p>
<p class="c1"><span class="c0">Code remains displayed</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a voltage issue</span></p>
<p class="c1"><span class="c0">It is not a sensor calibration error</span></p>
<p class="c1"><span class="c0">It is not a communication fault</span></p>
