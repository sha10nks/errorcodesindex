<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(ER PS &mdash; Power supply issue)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the appliance detected an internal power supply irregularity.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">ER PS appears when internal voltage regulation components detect abnormal power behavior. The appliance monitors input stability and internal distribution circuits. If voltage falls outside operating thresholds, the system pauses to protect components. This code reflects electrical safeguarding logic. It does not automatically confirm permanent damage.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">During startup</span></p>
<p class="c1"><span class="c0">After power fluctuation</span></p>
<p class="c1"><span class="c0">On display panel</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Voltage instability detected</span></p>
<p class="c1"><span class="c0">Internal regulation mismatch</span></p>
<p class="c1"><span class="c0">Protective threshold exceeded</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Operation pauses</span></p>
<p class="c1"><span class="c0">Appliance awaits reset</span></p>
<p class="c1"><span class="c0">Code remains displayed</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a cooling error</span></p>
<p class="c1"><span class="c0">It is not a communication failure</span></p>
<p class="c1"><span class="c0">It is not a sensor fault</span></p>
<p class="c1"><span class="c0">Related error codes</span></p>
<p class="c1"><span class="c0">PF &mdash; Power failure</span></p>
