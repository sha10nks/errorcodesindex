<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(E4 &mdash; Overflow condition)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the appliance detected excessive water levels.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">E4 appears when internal water level sensors detect levels beyond programmed safety thresholds. The appliance monitors fill levels continuously. If readings exceed safe limits, filling stops and draining may activate. This code reflects overflow protection logic. It does not automatically confirm permanent system damage.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">During fill stage</span></p>
<p class="c1"><span class="c0">Mid-cycle</span></p>
<p class="c1"><span class="c0">On the display panel</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Water level exceeded threshold</span></p>
<p class="c1"><span class="c0">Sensor reading outside range</span></p>
<p class="c1"><span class="c0">Protective overflow triggered</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Fill stops</span></p>
<p class="c1"><span class="c0">Drain may activate</span></p>
<p class="c1"><span class="c0">Cycle pauses</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a drain blockage</span></p>
<p class="c1"><span class="c0">It is not a heating fault</span></p>
<p class="c1"><span class="c0">It is not a motor imbalance</span></p>
<p class="c1"><span class="c0">Related error codes</span></p>
