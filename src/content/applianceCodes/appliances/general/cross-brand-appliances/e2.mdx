<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(E2 &mdash; Heating issue)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the appliance detected a heating irregularity.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">E2 appears when internal temperature monitoring does not detect expected heating performance. Sensors compare temperature changes against programmed targets. If thresholds are not reached within specified time limits, the cycle pauses. This code reflects heating system monitoring. It does not automatically confirm heating element failure.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">During wash or drying cycle</span></p>
<p class="c1"><span class="c0">Mid-operation</span></p>
<p class="c1"><span class="c0">On the display</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Temperature rise not detected</span></p>
<p class="c1"><span class="c0">Heating duration exceeded limit</span></p>
<p class="c1"><span class="c0">Sensor reading inconsistent</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Heating stops</span></p>
<p class="c1"><span class="c0">Cycle pauses</span></p>
<p class="c1"><span class="c0">Display continues showing code</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a drain problem</span></p>
<p class="c1"><span class="c0">It is not a motor imbalance</span></p>
<p class="c1"><span class="c0">It is not a water inlet issue</span></p>
<p class="c1"><span class="c0">Related error codes</span></p>
<p class="c1"><span class="c0">tE &mdash; Temperature sensor error</span></p>
<p class="c1"><span class="c0">E09 &mdash; Heating circuit failure</span></p>
<p class="c1"><span class="c0">F3 &mdash; Temperature sensor fault</span></p>
<p class="c1"><span class="c0">Notes and edge cases</span></p>
<p class="c1"><span class="c0">Cold incoming water may extend heating time.</span></p>
<p class="c1"><span class="c0">Optional next actions</span></p>
<p class="c1"><span class="c0">Restart cycle</span></p>
<p class="c1"><span class="c0">Observe temperature performance</span></p>
<p class="c1"><span class="c0">Monitor recurrence</span></p>
<p class="c1"><span class="c0">External help (optional)</span></p>
<p class="c1"><span class="c0">Ongoing heating alerts may require professional evaluation.</span></p>
