<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(E1 &mdash; Sensor failure)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the appliance detected an irregular sensor reading.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">E1 appears when internal monitoring detects a signal outside expected parameters from a sensor component. Appliances rely on multiple sensors for temperature, water level, and operation confirmation. If a signal becomes inconsistent or absent, the control system pauses the cycle. This code reflects a general sensor monitoring alert. It does not specify which component has failed.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">During active cycle</span></p>
<p class="c1"><span class="c0">On startup</span></p>
<p class="c1"><span class="c0">On the control display</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Sensor reading outside range</span></p>
<p class="c1"><span class="c0">Communication interruption</span></p>
<p class="c1"><span class="c0">Monitoring threshold exceeded</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Cycle pauses</span></p>
<p class="c1"><span class="c0">Certain functions may disable</span></p>
<p class="c1"><span class="c0">Code remains displayed</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not specifically a heating failure</span></p>
<p class="c1"><span class="c0">It is not necessarily a control board issue</span></p>
<p class="c1"><span class="c0">It is not a drain blockage</span></p>
<p class="c1"><span class="c0">Related error codes</span></p>
<p class="c1"><span class="c0">tE &mdash; Temperature sensor error</span></p>
<p class="c1"><span class="c0">E6 &mdash; Communication fault</span></p>

<p class="c1"><span class="c0">Temporary signal disruptions may occasionally trigger this alert.</span></p>
