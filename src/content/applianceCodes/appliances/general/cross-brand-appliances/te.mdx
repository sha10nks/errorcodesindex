<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(tE &mdash; Temperature sensor error)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the appliance detected irregular temperature sensor readings.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">tE appears when internal temperature sensors report readings outside expected parameters. The appliance compares temperature changes against programmed targets. If feedback is inconsistent or absent, the cycle pauses. This code reflects sensor monitoring safeguards. It does not automatically confirm heating element failure.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">During heating cycles</span></p>
<p class="c1"><span class="c0">Mid-operation</span></p>
<p class="c1"><span class="c0">On the display panel</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Sensor reading outside range</span></p>
<p class="c1"><span class="c0">Temperature feedback interruption</span></p>
<p class="c1"><span class="c0">Monitoring threshold exceeded</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Heating may stop</span></p>
<p class="c1"><span class="c0">Cycle pauses</span></p>
<p class="c1"><span class="c0">Display continues showing code</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a drain issue</span></p>
<p class="c1"><span class="c0">It is not a power failure</span></p>
<p class="c1"><span class="c0">It is not a motor malfunction</span></p>
<p class="c1"><span class="c0">Related error codes</span></p>
<p class="c1"><span class="c0">E2 &mdash; Heating issue</span></p>
<p class="c1"><span class="c0">E09 &mdash; Heating circuit failure</span></p>

<p class="c1"><span class="c0">Optional next actions</span></p>
