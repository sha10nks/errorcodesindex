<p class="c1"><span class="c0">CODE: dE</span></p>
<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(dE &mdash; Door error)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the appliance detected an issue with door closure or latch confirmation.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">dE appears when the appliance cannot confirm that the door is securely closed. Sensors monitor latch engagement before allowing operation. If the lock signal is absent or inconsistent, the cycle does not begin. This code reflects a safety interlock response. It does not automatically indicate structural damage.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">At cycle start</span></p>
<p class="c1"><span class="c0">After pressing start</span></p>
<p class="c1"><span class="c0">On the control display</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Door not fully closed</span></p>
<p class="c1"><span class="c0">Latch confirmation signal missing</span></p>
<p class="c1"><span class="c0">Safety timeout triggered</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Cycle does not begin</span></p>
<p class="c1"><span class="c0">Appliance remains idle</span></p>
<p class="c1"><span class="c0">Code stays visible</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a control board failure</span></p>
<p class="c1"><span class="c0">It is not a water inlet issue</span></p>
<p class="c1"><span class="c0">It is not a heating error</span></p>
<p class="c1"><span class="c0">Related error codes</span></p>
<p class="c1"><span class="c0">F22 &mdash; Door lock error</span></p>
<p class="c1"><span class="c0">F5 &mdash; Door latch problem</span></p>
<p class="c1"><span class="c0">LOC &mdash; Panel locked</span></p>
<p class="c1"><span class="c0">Notes and edge cases</span></p>
<p class="c1"><span class="c0">Misalignment may cause intermittent alerts.</span></p>
<p class="c1"><span class="c0">Optional next actions</span></p>
<p class="c1"><span class="c0">Ensure door is fully closed</span></p>
<p class="c1"><span class="c0">Restart cycle</span></p>
