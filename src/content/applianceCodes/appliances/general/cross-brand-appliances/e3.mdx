<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(E3 &mdash; Load or balance error)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the appliance detected an imbalance condition.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">E3 appears when rotation sensors detect excessive vibration or uneven load distribution. The appliance monitors balance during spin cycles. If imbalance exceeds safe thresholds, the cycle pauses to prevent strain. This code reflects stabilization protection logic. It does not indicate motor damage.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">During spin cycle</span></p>
<p class="c1"><span class="c0">Mid-cycle pause</span></p>
<p class="c1"><span class="c0">On the display panel</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Uneven load distribution</span></p>
<p class="c1"><span class="c0">Excessive vibration detected</span></p>
<p class="c1"><span class="c0">Balance threshold exceeded</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Spin slows or stops</span></p>
<p class="c1"><span class="c0">Cycle pauses</span></p>
<p class="c1"><span class="c0">System may attempt redistribution</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a drain issue</span></p>
<p class="c1"><span class="c0">It is not a heating malfunction</span></p>
<p class="c1"><span class="c0">It is not a control board error</span></p>
<p class="c1"><span class="c0">Related error codes</span></p>
<p class="c1"><span class="c0">UE &mdash; Unbalanced load</span></p>
<p class="c1"><span class="c0">LE &mdash; Motor locked</span></p>
<p class="c1"><span class="c0">F06 &mdash; Motor control error</span></p>
<p class="c1"><span class="c0">Notes and edge cases</span></p>
<p class="c1"><span class="c0">Bulky items may temporarily trigger imbalance detection.</span></p>
<p class="c1"><span class="c0">Optional next actions</span></p>
<p class="c1"><span class="c0">Redistribute items</span></p>
<p class="c1"><span class="c0">Restart spin</span></p>
<p class="c1"><span class="c0">Monitor recurrence</span></p>
