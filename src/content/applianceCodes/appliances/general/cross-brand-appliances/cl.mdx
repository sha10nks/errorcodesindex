<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(CL &mdash; Control lock active)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the appliance&rsquo;s control lock feature is currently enabled.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">CL appears when the control interface has been intentionally locked to prevent unintended input. Many appliances include a control lock or child lock feature that disables button functions. When activated, the display indicates the locked state. This code reflects a user-interface status rather than a malfunction. The appliance&rsquo;s primary systems typically remain operational if already running.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">On the control panel</span></p>
<p class="c1"><span class="c0">After pressing buttons</span></p>
<p class="c1"><span class="c0">When attempting to start a cycle</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Control lock feature activated</span></p>
<p class="c1"><span class="c0">Interface protection enabled</span></p>
<p class="c1"><span class="c0">Lock function engaged during prior use</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Buttons remain unresponsive</span></p>
<p class="c1"><span class="c0">Display continues to show lock status</span></p>
<p class="c1"><span class="c0">Existing cycle may continue</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a door lock failure</span></p>
<p class="c1"><span class="c0">It is not a control board error</span></p>
<p class="c1"><span class="c0">It is not a system malfunction</span></p>
<p class="c1"><span class="c0">Related error codes</span></p>
<p class="c1"><span class="c0">LOC &mdash; Panel locked</span></p>
<p class="c1"><span class="c0">F22 &mdash; Door lock error</span></p>
<p class="c1"><span class="c0">dE &mdash; Door error</span></p>
<p class="c1"><span class="c0">Notes and edge cases</span></p>
<p class="c1"><span class="c0">Some models automatically enable control lock after certain combinations of inputs.</span></p>
<p class="c1"><span class="c0">Optional next actions</span></p>
<p class="c1"><span class="c0">Disable control lock feature</span></p>
<p class="c1"><span class="c0">Verify user settings</span></p>
