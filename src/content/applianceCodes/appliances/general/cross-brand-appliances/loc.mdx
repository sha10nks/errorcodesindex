<p class="c1"><span class="c0">CODE: LOC</span></p>
<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(LOC &mdash; Panel locked)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the appliance control panel is locked.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">LOC is displayed when the appliance&rsquo;s interface lock function has been activated. This feature prevents unintended changes to cycle settings. The appliance continues monitoring system operations normally while the panel remains locked. The code represents a user-interface state. It does not indicate mechanical or electrical failure.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">On the display panel</span></p>
<p class="c1"><span class="c0">After pressing control buttons</span></p>
<p class="c1"><span class="c0">During normal operation</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Panel lock feature enabled</span></p>
<p class="c1"><span class="c0">Safety lock activated</span></p>
<p class="c1"><span class="c0">Input protection setting engaged</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Controls remain disabled</span></p>
<p class="c1"><span class="c0">Active cycle may continue</span></p>
<p class="c1"><span class="c0">Display shows lock indicator</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a door latch error</span></p>
<p class="c1"><span class="c0">It is not a control board malfunction</span></p>
<p class="c1"><span class="c0">It is not a power supply issue</span></p>
<p class="c1"><span class="c0">Related error codes</span></p>
<p class="c1"><span class="c0">CL &mdash; Control lock active</span></p>
<p class="c1"><span class="c0">F5 &mdash; Door latch problem</span></p>
<p class="c1"><span class="c0">dE &mdash; Door error</span></p>
<p class="c1"><span class="c0">Notes and edge cases</span></p>
<p class="c1"><span class="c0">Some appliances automatically lock panels during high-temperature cycles.</span></p>
<p class="c1"><span class="c0">Optional next actions</span></p>
<p class="c1"><span class="c0">Disable panel lock feature</span></p>
<p class="c1"><span class="c0">Restart appliance if needed</span></p>
