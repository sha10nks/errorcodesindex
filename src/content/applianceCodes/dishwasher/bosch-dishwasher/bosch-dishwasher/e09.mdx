<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(E09 &mdash; Heating circuit failure)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the dishwasher detected an issue within the heating system.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">E09 appears when internal sensors detect that the heating circuit is not functioning within expected parameters. The appliance monitors temperature progression during wash cycles. If heating thresholds are not achieved, the cycle may stop. This code reflects heating system monitoring. It does not automatically confirm element damage.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">During wash or drying stage</span></p>
<p class="c1"><span class="c0">Mid-cycle interruption</span></p>
<p class="c1"><span class="c0">On the digital display</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Temperature increase not detected</span></p>
<p class="c1"><span class="c0">Heating time exceeded limit</span></p>
<p class="c1"><span class="c0">Sensor feedback irregular</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Cycle pauses</span></p>
<p class="c1"><span class="c0">Drying performance reduced</span></p>
<p class="c1"><span class="c0">Appliance remains powered</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a drain blockage</span></p>
<p class="c1"><span class="c0">It is not a leak alert</span></p>
<p class="c1"><span class="c0">It is not a motor fault</span></p>
<p class="c1"><span class="c0">Related error codes</span></p>
<p class="c1"><span class="c0">HE &mdash; Heating error</span></p>
<p class="c1"><span class="c0">E2 &mdash; Heating issue</span></p>
<p class="c1"><span class="c0">ER DH &mdash; Defrost heater error</span></p>
<p class="c1"><span class="c0">Notes and edge cases</span></p>
<p class="c1"><span class="c0">Lower water temperatures may influence sensor readings in certain cycles.</span></p>
<p class="c1"><span class="c0">Optional next actions</span></p>
<p class="c1"><span class="c0">Restart cycle</span></p>
<p class="c1"><span class="c0">Observe heating performance</span></p>
