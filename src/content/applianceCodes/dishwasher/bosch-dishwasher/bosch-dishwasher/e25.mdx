<p class="c1"><span class="c0">Title</span></p>
<p class="c1"><span class="c0">(E25 &mdash; Drain pump blocked)</span></p>
<p class="c1"><span class="c0">One-sentence summary</span></p>
<p class="c1"><span class="c0">This code indicates that the dishwasher detected restricted pump rotation or flow.</span></p>
<p class="c1"><span class="c0">What this code means</span></p>
<p class="c1"><span class="c0">E25 appears when the appliance detects that the drain pump is not operating within expected parameters. The system monitors pump performance during water removal. If flow or rotation feedback falls outside programmed limits, the cycle stops. This code reflects pump monitoring logic. It does not automatically confirm permanent pump failure.</span></p>
<p class="c1"><span class="c0">Where users usually see this code</span></p>
<p class="c1"><span class="c0">During the drain phase</span></p>
<p class="c1"><span class="c0">At cycle completion</span></p>
<p class="c1"><span class="c0">On the display panel</span></p>
<p class="c1"><span class="c0">Why this code usually appears</span></p>
<p class="c1"><span class="c0">Drain flow below expected level</span></p>
<p class="c1"><span class="c0">Pump rotation irregularity detected</span></p>
<p class="c1"><span class="c0">Protective threshold exceeded</span></p>
<p class="c1"><span class="c0">What typically happens next</span></p>
<p class="c1"><span class="c0">Cycle pauses</span></p>
<p class="c1"><span class="c0">Water may remain inside</span></p>
<p class="c1"><span class="c0">Appliance remains powered but inactive</span></p>
<p class="c1"><span class="c0">What this code is NOT</span></p>
<p class="c1"><span class="c0">It is not a heating error</span></p>
<p class="c1"><span class="c0">It is not a fill issue</span></p>
<p class="c1"><span class="c0">It is not a control system malfunction</span></p>
<p class="c1"><span class="c0">Related error codes</span></p>
<p class="c1"><span class="c0">E24 &mdash; Drain error</span></p>
<p class="c1"><span class="c0">OE &mdash; Drain error</span></p>
